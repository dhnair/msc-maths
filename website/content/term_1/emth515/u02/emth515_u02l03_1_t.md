## Learning Outcomes
- Prove linearity with respect to the integrator:  
  If \( f \in R(lpha_1) \) and \( f \in R(lpha_2) \), then \( f \in R(lpha_1+lpha_2) \).
- Establish integral additivity:  
  \( \int_a^b f\, d(lpha_1+lpha_2) = \int_a^b f\, dlpha_1 + \int_a^b f\, dlpha_2. \)
- Prove monotonicity of the RS integral:  
  If \( f \ge 0 \), then \( \int_a^b f\, dlpha \ge 0 \).
- Prove order preservation:  
  If \( f_1 \le f_2 \), then \( \int_a^b f_1\, dlpha \le \int_a^b f_2\, dlpha \).

## Content

### 1. Linearity in the Integrator
Given:
- \( f \in R(lpha_1) \),
- \( f \in R(lpha_2) \).

For a given \( arepsilon > 0 \), choose partitions \( P_1, P_2 \) such that:
\[
U(P_1, f, lpha_1) - L(P_1, f, lpha_1) < arepsilon/2,
\]
\[
U(P_2, f, lpha_2) - L(P_2, f, lpha_2) < arepsilon/2.
\]

Let \( P = P_1 \cup P_2 \), the common refinement. Then:
\[
U(P,f,lpha_1) - L(P,f,lpha_1) < arepsilon/2,
\]
\[
U(P,f,lpha_2) - L(P,f,lpha_2) < arepsilon/2.
\]

Take \( lpha = lpha_1 + lpha_2 \). Bounds yield:
\[
L(P,f,lpha_1) + L(P,f,lpha_2) \le L(P,f,lpha) \le U(P,f,lpha)
\le U(P,f,lpha_1) + U(P,f,lpha_2).
\]

Thus:
\[
U(P,f,lpha) - L(P,f,lpha) < arepsilon.
\]
Hence \( f \in R(lpha_1 + lpha_2) \).

### 2. Additivity of the Integral
Using the inequalities:
\[
\int_a^b f\, dlpha_1 \le U(P,f,lpha_1), \quad
\int_a^b f\, dlpha_2 \le U(P,f,lpha_2),
\]
and lower-sum relations, one obtains:
\[
\int_a^b f\, d(lpha_1 + lpha_2)
= \int_a^b f\, dlpha_1 + \int_a^b f\, dlpha_2.
\]

### 3. Non-negativity Property
If \( f(x) \ge 0 \) for all \( x \in [a,b] \):
- Then \( m_i \ge 0 \) on each subinterval.
- So \( L(P,f,lpha) \ge 0 \) for every partition.
- Therefore:
\[
\int_a^b f\, dlpha \ge 0.
\]

### 4. Order Preservation
If \( f_1 \le f_2 \) on \( [a,b] \), then \( f_2 - f_1 \ge 0 \).  
Since both functions are RS-integrable:
\[
\int_a^b (f_2 - f_1)\, dlpha \ge 0
\quad \Rightarrow \quad
\int_a^b f_2\, dlpha \ge \int_a^b f_1\, dlpha.
\]

## Summary
This lecture established key linearity and monotonicity properties of the RS integral:
- Linear with respect to the integrator.
- Integrals add when integrators add.
- Non-negativity when the function is non-negative.
- Order preservation between functions.
