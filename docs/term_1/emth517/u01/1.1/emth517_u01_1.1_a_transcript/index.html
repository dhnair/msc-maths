Introduction to Differential Equations (Transcript)

Learning Outcomes​

After this lecture, you will be able to:

  • Understand why differential equations are used in modeling.
  • Identify order and normal form of an ODE.
  • Recognize linear and nonlinear equations.
  • Understand initial value problems.

1. Motivation​

Many physical, biological, and engineering systems involve rates of change.
Differential equations capture these relationships using:

  • independent variable t
  • dependent variable u(t)
  • derivatives u'(t), u''(t), and so on

3. Order and Form​

An nth order ODE has the form:

F(t,u,u′,u′′,…,u(n))=0F(t, u, u', u'', \ldots, u^{(n)}) = 0F(t,u,u′,u′′,…,u(n))=0

If it can be written as:

u(n)=g(t,u,u′,…,u(n−1))u^{(n)} = g(t, u, u', \ldots, u^{(n-1)})u(n)=g(t,u,u′,…,u(n−1))

it is said to be in normal form.

5. Linear vs Nonlinear​

A first order ODE,

u′=g(t,u)u' = g(t,u)u′=g(t,u)

is linear if g(t,u) is linear in u.
Otherwise it is nonlinear.

Example linear ODE:

u′=tu+2tu' = t u + 2 tu′=tu+2t

Its general solution is:

u(t)=cet2/2−2u(t) = c e^{t^2 / 2} - 2u(t)=cet2/2−2

6. Initial Value Problem (IVP)​

An IVP consists of an ODE and an initial condition:

u(t0)=u0u(t_0) = u_0u(t0​)=u0​

A function u(t) solves the IVP if:

  • it is differentiable,
  • satisfies the ODE,
  • satisfies the initial condition,
  • remains inside the domain of definition.

Summary​

Differential equations describe systems where change is essential.
This lecture introduced order, form, linearity, and initial value problems.
