Definition and Existence of the Integral (Transcript)

Learning Outcomes​

After this lecture, you will be able to:

  • Recall the definition of a bounded function.
  • Understand the concepts of partition, mesh, infimum, and supremum.
  • Define and compute lower and upper Riemann sums.
  • Understand conditions for Riemann integrability.
  • Extend the idea to the Riemann–Stieltjes integral.
  • Recognize that the Riemann integral is a special case of the Riemann–Stieltjes integral.

1. Introduction​

This lecture introduces the Riemann–Stieltjes integral, its definition, existence, and properties.
We begin with the classical Riemann integral before extending it.

2. Definitions and Setup​

  • Let f:[a,b]→Rf:[a,b] \to \mathbb{R}f:[a,b]→R be a bounded function.

Definition 2.1 (Bounded Function).
A function fff is bounded on [a,b][a,b][a,b] if there exist real numbers m,Mm, Mm,M such that:

m≤f(x)≤M,∀x∈[a,b].m \leq f(x) \leq M, \quad \forall x \in [a,b].m≤f(x)≤M,∀x∈[a,b].
  • A partition PPP of [a,b][a,b][a,b] is:
P={x0,x1,…,xn},a=x0<x1<⋯<xn=b.P = \{x_0, x_1, \dots, x_n\}, \quad a=x_0 < x_1 < \dots < x_n=b.P={x0​,x1​,…,xn​},a=x0​<x1​<⋯<xn​=b.

Definition 2.2 (Mesh).
For each subinterval:

Δxi=xi−xi−1,i=1,2,…,n.\Delta x_i = x_i - x_{i-1}, \quad i=1,2,\dots,n.Δxi​=xi​−xi−1​,i=1,2,…,n.

Clearly, Δxi≥0\Delta x_i \geq 0Δxi​≥0, and:

∑i=1nΔxi=b−a.\sum_{i=1}^n \Delta x_i = b-a.i=1∑n​Δxi​=b−a.

Definition 2.3 (Infimum and Supremum).
For each subinterval [xi−1,xi][x_{i-1},x_i][xi−1​,xi​]:

  • mi=inf⁡{f(x):x∈[xi−1,xi]}m_i = \inf \{f(x): x \in [x_{i-1},x_i]\}mi​=inf{f(x):x∈[xi−1​,xi​]}
  • Mi=sup⁡{f(x):x∈[xi−1,xi]}M_i = \sup \{f(x): x \in [x_{i-1},x_i]\}Mi​=sup{f(x):x∈[xi−1​,xi​]}

Relation:

m≤mi≤Mi≤M.m \leq m_i \leq M_i \leq M.m≤mi​≤Mi​≤M.

3. Riemann Sums​

Definition 3.1 (Lower and Upper Sums).

  • Lower sum:
L(P,f)=∑i=1nmiΔxiL(P,f) = \sum_{i=1}^n m_i \Delta x_iL(P,f)=i=1∑n​mi​Δxi​
  • Upper sum:
U(P,f)=∑i=1nMiΔxiU(P,f) = \sum_{i=1}^n M_i \Delta x_iU(P,f)=i=1∑n​Mi​Δxi​

Theorem 3.1 (Bounds on Riemann Sums).
For all partitions PPP,

m(b−a)≤L(P,f)≤U(P,f)≤M(b−a).m(b-a) \leq L(P,f) \leq U(P,f) \leq M(b-a).m(b−a)≤L(P,f)≤U(P,f)≤M(b−a).

Proof:
Since m≤mi≤Mi≤Mm \leq m_i \leq M_i \leq Mm≤mi​≤Mi​≤M and Δxi≥0\Delta x_i \geq 0Δxi​≥0, multiplying by Δxi\Delta x_iΔxi​ preserves inequalities. Summing over i=1i=1i=1 to nnn yields the stated inequality.
□

4. Riemann Integral​

Definition 4.1 (Lower Riemann Integral).

∫ab‾f(x) dx=sup⁡PL(P,f)\underline{\int_a^b} f(x)\,dx = \sup_{P} L(P,f)∫ab​​f(x)dx=Psup​L(P,f)

Definition 4.2 (Upper Riemann Integral).

∫ab‾f(x) dx=inf⁡PU(P,f)\overline{\int_a^b} f(x)\,dx = \inf_{P} U(P,f)∫ab​​f(x)dx=Pinf​U(P,f)

Definition 4.3 (Riemann Integrability).
If ∫ab‾f(x) dx=∫ab‾f(x) dx\underline{\int_a^b} f(x)\,dx = \overline{\int_a^b} f(x)\,dx∫ab​​f(x)dx=∫ab​​f(x)dx, we define:

∫abf(x) dx\int_a^b f(x)\,dx∫ab​f(x)dx

and say fff is Riemann integrable.

5. Refinement of Partitions​

  • If P∗⊇PP^* \supseteq PP∗⊇P, then P∗P^*P∗ is a refinement of PPP.
  • If P=P1∪P2P = P_1 \cup P_2P=P1​∪P2​, then PPP is the common refinement of P1,P2P_1, P_2P1​,P2​.

6. Riemann–Stieltjes Integral​

Definition 6.1 (Riemann–Stieltjes Integral).
Let f:[a,b]→Rf:[a,b] \to \mathbb{R}f:[a,b]→R be bounded, and let α:[a,b]→R\alpha:[a,b]\to\mathbb{R}α:[a,b]→R be monotonically increasing.
Define:

Δαi=α(xi)−α(xi−1)≥0.\Delta \alpha_i = \alpha(x_i) - \alpha(x_{i-1}) \geq 0.Δαi​=α(xi​)−α(xi−1​)≥0.
  • Lower sum:
L(P,f,α)=∑i=1nmiΔαiL(P,f,\alpha) = \sum_{i=1}^n m_i \Delta \alpha_iL(P,f,α)=i=1∑n​mi​Δαi​
  • Upper sum:
U(P,f,α)=∑i=1nMiΔαiU(P,f,\alpha) = \sum_{i=1}^n M_i \Delta \alpha_iU(P,f,α)=i=1∑n​Mi​Δαi​

Theorem 6.1 (Bounds on Riemann–Stieltjes Sums).
For all partitions PPP,

m(α(b)−α(a))≤L(P,f,α)≤U(P,f,α)≤M(α(b)−α(a)).m(\alpha(b)-\alpha(a)) \leq L(P,f,\alpha) \leq U(P,f,\alpha) \leq M(\alpha(b)-\alpha(a)).m(α(b)−α(a))≤L(P,f,α)≤U(P,f,α)≤M(α(b)−α(a)).

Proof:
Same reasoning as Theorem 3.1, but using Δαi≥0\Delta \alpha_i \geq 0Δαi​≥0 since α\alphaα is monotone increasing.
□

Definition 6.2 (Riemann–Stieltjes Integrability).

  • Lower integral:
∫ab‾f(x) dα(x)=sup⁡PL(P,f,α)\underline{\int_a^b} f(x)\,d\alpha(x) = \sup_{P} L(P,f,\alpha)∫ab​​f(x)dα(x)=Psup​L(P,f,α)
  • Upper integral:
∫ab‾f(x) dα(x)=inf⁡PU(P,f,α)\overline{\int_a^b} f(x)\,d\alpha(x) = \inf_{P} U(P,f,\alpha)∫ab​​f(x)dα(x)=Pinf​U(P,f,α)

If equal, then:

∫abf(x) dα(x)\int_a^b f(x)\,d\alpha(x)∫ab​f(x)dα(x)

exists, and fff is Riemann–Stieltjes integrable.

7. Special Case​

If α(x)=x\alpha(x) = xα(x)=x, then:

∫abf(x) dα(x)=∫abf(x) dx\int_a^b f(x)\,d\alpha(x) = \int_a^b f(x)\,dx∫ab​f(x)dα(x)=∫ab​f(x)dx

Thus the Riemann integral is a special case of the Riemann–Stieltjes integral.

8. Summary​

  • Defined bounded function, partition, infimum, supremum.
  • Introduced lower and upper Riemann sums.
  • Defined Riemann integrability.
  • Extended to Riemann–Stieltjes integral using monotone increasing α\alphaα.
  • Riemann integral = special case with α(x)=x\alpha(x)=xα(x)=x.

9. Footer: Glossary & Notation​

Glossary​

  • Bounded Function: f:[a,b]→Rf:[a,b]\to\mathbb{R}f:[a,b]→R is bounded if ∃m,M∈R\exists m,M \in \mathbb{R}∃m,M∈R with m≤f(x)≤Mm \leq f(x)\leq Mm≤f(x)≤M for all x∈[a,b]x\in[a,b]x∈[a,b].
  • Partition: A finite sequence a=x0<x1<⋯<xn=ba=x_0<x_1<\dots<x_n=ba=x0​<x1​<⋯<xn​=b.
  • Mesh: Δxi=xi−xi−1\Delta x_i = x_i-x_{i-1}Δxi​=xi​−xi−1​.
  • Infimum (inf): Greatest lower bound of a set.
  • Supremum (sup): Least upper bound of a set.
  • Riemann Integral: Defined using limits of sums over partitions.
  • Riemann–Stieltjes Integral: Generalization of Riemann integral with monotone increasing α\alphaα.

Notation​

  • R\mathbb{R}R = set of real numbers
  • N\mathbb{N}N = set of natural numbers
  • L(P,f)L(P,f)L(P,f) = lower Riemann sum
  • U(P,f)U(P,f)U(P,f) = upper Riemann sum
