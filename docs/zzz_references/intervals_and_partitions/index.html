Intervals and Partitions in Real Analysis

Learning Outcomes​

By the end of this note, you should be able to:

  1. Define and distinguish between open, closed, and half-open intervals.
  2. Represent intervals both formally (set-builder notation) and geometrically (number line).
  3. Understand the role of intervals in defining partitions.
  4. Explain what a partition of an interval is and why it is important for Riemann integration.

1. Intervals​

An interval is a subset of the real numbers R\mathbb{R}R consisting of all real numbers lying between two given numbers aaa and bbb (with a<ba < ba<b).

Closed Interval​

[a,b]={x∈R:a≤x≤b}.[a,b] = \{ x \in \mathbb{R} : a \leq x \leq b \}.[a,b]={x∈R:a≤x≤b}.
  • Includes both endpoints.
  • Example: [0,1]={x:0≤x≤1}[0,1] = \{ x : 0 \leq x \leq 1 \}[0,1]={x:0≤x≤1}.

Open Interval​

(a,b)={x∈R:a<x<b}.(a,b) = \{ x \in \mathbb{R} : a < x < b \}.(a,b)={x∈R:a<x<b}.
  • Excludes both endpoints.
  • Example: (0,1)={x:0<x<1}(0,1) = \{ x : 0 < x < 1 \}(0,1)={x:0<x<1}.

Half-Open Intervals​

[a,b)={x:a≤x<b},(a,b]={x:a<x≤b}.[a,b) = \{ x : a \leq x < b \}, \quad (a,b] = \{ x : a < x \leq b \}.[a,b)={x:a≤x<b},(a,b]={x:a<x≤b}.
  • Includes only one endpoint.

2. Partitions of Intervals​

Definition: A partition PPP of an interval [a,b][a,b][a,b] is a finite sequence of points

P={x0,x1,…,xn}P = \{x_0, x_1, \dots, x_n\}P={x0​,x1​,…,xn​}

such that

a=x0<x1<x2<⋯<xn−1<xn=b.a = x_0 < x_1 < x_2 < \dots < x_{n-1} < x_n = b.a=x0​<x1​<x2​<⋯<xn−1​<xn​=b.

  • The points divide the interval [a,b][a,b][a,b] into subintervals

    [x0,x1],[x1,x2],…,[xn−1,xn].[x_0, x_1], [x_1, x_2], \dots, [x_{n-1}, x_n].[x0​,x1​],[x1​,x2​],…,[xn−1​,xn​].

  • If instead we considered an open interval (a,b)(a,b)(a,b), then the partition would satisfy

    a<x0<x1<⋯<xn<b.a < x_0 < x_1 < \dots < x_n < b.a<x0​<x1​<⋯<xn​<b.

    But in Riemann integration, we almost always work with closed intervals [a,b][a,b][a,b].

3. Why Partitions Matter​

  • Riemann Integration: A Riemann sum is formed by choosing sample points in each subinterval of a partition, then summing up rectangles that approximate the area under a curve.
  • Flexibility: Different partitions lead to different Riemann sums, but as partitions become finer, these sums approach the integral.
  • Endpoint Inclusion: The use of closed intervals [a,b][a,b][a,b] ensures that both boundary points are included, avoiding ambiguity.

Glossary​

  • Interval: A set of real numbers between two endpoints.
  • Closed interval: [a,b][a,b][a,b], includes both endpoints.
  • Open interval: (a,b)(a,b)(a,b), excludes both endpoints.
  • Half-open interval: Includes one endpoint, excludes the other.
  • Partition: A finite ordered set of points dividing an interval into smaller subintervals.
  • Subinterval: A smaller interval formed by consecutive points of a partition.
  • Riemann sum: Approximation of an integral using a partition and sample points.

  • Intervals may be open, closed, or half-open depending on whether endpoints are included.

  • Partitions are the foundation for defining Riemann sums and integrals.
