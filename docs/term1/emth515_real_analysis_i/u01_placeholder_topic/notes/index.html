🧮 Quick Revision Notes: Riemann–Stieltjes Integral

Key Concepts​

Let a partition be denoted as P={x0,x1,…,xn}P = \{x_0, x_1, \dots, x_n\}P={x0​,x1​,…,xn​}
where a=x0a = x_0a=x0​ and b=xnb = x_nb=xn​.

For each subinterval [xi−1,xi][x_{i-1}, x_i][xi−1​,xi​]:

  • mi=inf⁡{f(x):x∈[xi−1,xi]}m_i = \inf\{ f(x) : x \in [x_{i-1}, x_i] \}mi​=inf{f(x):x∈[xi−1​,xi​]}
  • Mi=sup⁡{f(x):x∈[xi−1,xi]}M_i = \sup\{ f(x) : x \in [x_{i-1}, x_i] \}Mi​=sup{f(x):x∈[xi−1​,xi​]}
  • Δxi=xi−xi−1≥0\Delta x_i = x_i - x_{i-1} \ge 0Δxi​=xi​−xi−1​≥0

Riemann Sums​

  • Lower Sum:
    L(P,f)=∑imi ΔxiL(P, f) = \sum_i m_i \, \Delta x_iL(P,f)=∑i​mi​Δxi​

  • Upper Sum:
    U(P,f)=∑iMi ΔxiU(P, f) = \sum_i M_i \, \Delta x_iU(P,f)=∑i​Mi​Δxi​

  • Relation:
    m(b−a)≤L(P,f)≤U(P,f)≤M(b−a)m(b - a) \le L(P, f) \le U(P, f) \le M(b - a)m(b−a)≤L(P,f)≤U(P,f)≤M(b−a)

If sup⁡L(P,f)=inf⁡U(P,f)\sup L(P, f) = \inf U(P, f)supL(P,f)=infU(P,f),
then fff is Riemann integrable on ([a, b]$.

Riemann–Stieltjes Generalization​

Let α(x)\alpha(x)α(x) be a monotonically increasing function.
Define Δαi=α(xi)−α(xi−1)≥0\Delta \alpha_i = \alpha(x_i) - \alpha(x_{i-1}) \ge 0Δαi​=α(xi​)−α(xi−1​)≥0.

  • Lower Sum:
    L(P,f,α)=∑imi ΔαiL(P, f, \alpha) = \sum_i m_i \, \Delta \alpha_iL(P,f,α)=∑i​mi​Δαi​

  • Upper Sum:
    U(P,f,α)=∑iMi ΔαiU(P, f, \alpha) = \sum_i M_i \, \Delta \alpha_iU(P,f,α)=∑i​Mi​Δαi​

  • Relation:
    m[α(b)−α(a)]≤L(P,f,α)≤U(P,f,α)≤M[α(b)−α(a)]m[\alpha(b) - \alpha(a)] \le L(P, f, \alpha) \le U(P, f, \alpha) \le M[\alpha(b) - \alpha(a)]m[α(b)−α(a)]≤L(P,f,α)≤U(P,f,α)≤M[α(b)−α(a)]

If sup⁡L(P,f,α)=inf⁡U(P,f,α)\sup L(P, f, \alpha) = \inf U(P, f, \alpha)supL(P,f,α)=infU(P,f,α),
then f∈R(α)f \in R(\alpha)f∈R(α), i.e. fff is Riemann–Stieltjes integrable with respect to α\alphaα.

Exam Tip 💡​

  • Always check that α(x)\alpha(x)α(x) is monotonic.
  • For step functions α(x)\alpha(x)α(x), integration reduces to summation.
  • The Riemann–Stieltjes integral exists only if the upper and lower sums coincide.
