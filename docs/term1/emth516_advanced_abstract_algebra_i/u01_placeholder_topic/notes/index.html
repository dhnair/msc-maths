EMTH516 – Unit 1: Review of Groups (Quick Revision)

Binary Operations​

A binary operation on a set SSS is a function ∗:S×S→S*: S \times S \to S∗:S×S→S.

Closed if a∗b∈Sa*b \in Sa∗b∈S for all a,b∈Sa,b \in Sa,b∈S.
think: result must stay inside the same set.

Examples:

  • (Z,+),(Z,×)(\mathbb{Z}, +), (\mathbb{Z}, \times)(Z,+),(Z,×) → closed.
  • (Z,/)(\mathbb{Z}, /)(Z,/) → not closed (e.g. 1/2∉Z1/2 \notin \mathbb{Z}1/2∈/Z).

Key properties:

  • Associative: (a∗b)∗c=a∗(b∗c)(a*b)*c = a*(b*c)(a∗b)∗c=a∗(b∗c)
  • Commutative: a∗b=b∗aa*b = b*aa∗b=b∗a
  • Identity: e∗a=a∗e=ae*a = a*e = ae∗a=a∗e=a
  • Inverse: a∗a−1=ea*a^{-1} = ea∗a−1=e

Algebraic Structures​

StructureConditions
SemigroupClosure + Associativity
MonoidSemigroup + Identity
GroupMonoid + Inverses
Abelian GroupGroup + Commutativity

Examples:

  • (Z,+)(\mathbb{Z}, +)(Z,+) → abelian group (identity 0, inverse −a-a−a).
  • (Z,×)(\mathbb{Z}, \times)(Z,×) → monoid (identity 1, most elements no inverse).
  • (even integers,×)(\text{even integers}, \times)(even integers,×) → semigroup only (1 not even).

note: group axioms build hierarchically.

Groups​

A group (G,∗)(G, *)(G,∗) satisfies:

  1. Closure: a∗b∈Ga*b \in Ga∗b∈G
  2. Associativity
  3. Identity e∈Ge \in Ge∈G
  4. Inverse a−1∈Ga^{-1} \in Ga−1∈G

Abelian Group: a∗b=b∗aa*b = b*aa∗b=b∗a.
Order of a Group: number of elements in GGG.

Order of an Element: smallest nnn s.t. an=ea^n = ean=e.

examples:

  • i∈(C∗,×)i \in (\mathbb{C}^*, \times)i∈(C∗,×) → order 4.
  • (Z6,+6)(\mathbb{Z}_6, +_6)(Z6​,+6​) → cyclic of order 6.

think: order → repetition until identity.

Subgroups​

H≤GH \leq GH≤G if HHH is non-empty and closed under group operation and inverses.

Examples:

  • (Z,+)(\mathbb{Z}, +)(Z,+) subgroup of (R,+)(\mathbb{R}, +)(R,+).
  • Even integers form subgroup of integers under addition.

Normal Subgroups & Cosets​

A subgroup HHH of GGG is normal (H⊴GH \trianglelefteq GH⊴G) if gH=HggH = HggH=Hg for all g∈Gg \in Gg∈G.

equivalent: g−1hg∈Hg^{-1}hg \in Hg−1hg∈H.

Coset: gH={gh∣h∈H}gH = \{gh | h \in H\}gH={gh∣h∈H}
Index: number of distinct cosets = [G:H][G : H][G:H].

Lagrange’s Theorem:
∣G∣=[G:H]∣H∣|G| = [G : H] |H|∣G∣=[G:H]∣H∣
order of subgroup divides order of group.

Examples:

  • In Z12\mathbb{Z}_{12}Z12​, subgroup {0,4,8}\{0,4,8\}{0,4,8} has index 4.

think: groups partitioned into equal-sized cosets.

Cyclic & Quotient Groups​

A group GGG is cyclic if G=⟨a⟩={an∣n∈Z}G = \langle a \rangle = \{a^n | n \in \mathbb{Z}\}G=⟨a⟩={an∣n∈Z}.

  • All cyclic groups are abelian.
  • Subgroups of cyclic groups are cyclic.

Quotient Group: If H⊴GH \trianglelefteq GH⊴G, then G/H={Hg:g∈G}G/H = \{Hg : g \in G\}G/H={Hg:g∈G} with (Ha)(Hb)=H(ab)(Ha)(Hb) = H(ab)(Ha)(Hb)=H(ab).

note: quotient = “group of cosets”.

Homomorphisms & Isomorphisms​

A function ϕ:(G,∗)→(G′,∘)\phi: (G, *) \to (G', \circ)ϕ:(G,∗)→(G′,∘) is a homomorphism if:
ϕ(a∗b)=ϕ(a)∘ϕ(b)\phi(a*b) = \phi(a) \circ \phi(b)ϕ(a∗b)=ϕ(a)∘ϕ(b)

  • Kernel: ker⁡(ϕ)={g∣ϕ(g)=e′}\ker(\phi) = \{g | \phi(g)=e'\}ker(ϕ)={g∣ϕ(g)=e′} → normal subgroup.
  • Image: Im(ϕ)={ϕ(g)∣g∈G}\text{Im}(\phi) = \{\phi(g) | g \in G\}Im(ϕ)={ϕ(g)∣g∈G} → subgroup.

Fundamental Theorem:
G/ker⁡(ϕ)≅Im(ϕ)G/\ker(\phi) \cong \text{Im}(\phi)G/ker(ϕ)≅Im(ϕ)

Isomorphism: bijective homomorphism.
think: same structure, different skin.
