# U02L07 - Combining Integrals and Composite Partitions (Notes)

## Key Points
- RS sum: S_P(f,α)=∑ f(tᵢ)Δαᵢ.
- Always bounded: LP ≤ S_P ≤ UP.
- If lim_{‖P‖→0} S_P exists, f ∈ R_α.
- Value of integral equals limit.
- If α' continuous, ∫ f dα = ∫ f α' dx.

## Definitions and Notation
- mᵢ, Mᵢ: interval infimum/supremum.
- Δαᵢ=α(xᵢ)−α(x_{i−1}).

## Core Results
- LP(f,α) ≤ S_P(f,α) ≤ UP(f,α).
- Convergence of sums ⇒ UP−LP < ε ⇒ integrability.
- Integral equals limit A.
- With α' continuous: Δαᵢ=α'(tᵢ)Δxᵢ.

## Important Derivations
- Bounds use total variation α(b)−α(a).
- Error ≤ ε/4 on sums used to squeeze UP−LP.

## Relationships
- RS integral generalizes Riemann integral; equality holds when integrator smooth.

## Summary
RS sum convergence gives integrability and the value of the integral; smooth α reduces RS integration to ordinary Riemann integration.
